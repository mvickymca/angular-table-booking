Example API Usage

# Upload a file using curl
curl -X POST http://localhost:8080/api/upload \
  -F "file=@example-data.json"

# Get JSON data by ID
curl http://localhost:8080/api/json/1

# Get API information
curl http://localhost:8080/api/documents
